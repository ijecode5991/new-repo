HTML TAGS

Firstly what is HTML? HTML is the acronym for HYPERTEXT MARKUP LANGUAGE. It allows the user to create and structure sections, paragraphs, headings, links, and blockquotes for webpages and applications.

HTML is not a programming language. When working with HTML, we use simple code structures (tags and attributes) to markup a website page.

HOW DOES HTML WORK?

HTML document are files that end with a .html or .htm extention. You can view then by using any web browser (such as Google Chrome, Safari, or Mozilla Firefox). The browser reads the HTML file and renders its content so that internet users can view it.

HTML TAGS

There are several HTML TAGS that are used in creating or structring a webpage. Examples of these tags used are:

    Image tag
    Link tag
    Paragraph tag
    Heading tag

The image tag "< img>". The img element lets you insert images into your webpages. To insert an image, you first upload the image to your web server, then us an < img> tag to reference the uploaded image filename.Example is:

< img src= "myphoto.jpg" alt= "My Photo">




The link Tag "< a>". One of the most important elements in a web page, the a element lets you create links to the other content. The content can be either on your own site or on another site.
To create a link, you wrap < a> and < /a> tags around the content you want to use for the link, and supply the URL to link to in the < a> tags href attribute.

Here's how to create some text that links to www.example.com: < a href= "http://www.example.com">Visit this great website < /a>
